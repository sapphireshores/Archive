# Archive 

A minimalist, website deployed to serve as a portfolio for my technical projects and milestones.

## Purpose & Architecture
Rather than using abstract content management systems or heavy JavaScript frameworks, this single-page deployment is made using clean semantic markup and structural grid styling. It acts as the primary access portal for my projects,.

## Infrastructure & Tech Stack
* **Markup & Core Structure:** HTML5 (Semantic formatting)
* **Styling Framework:** Bootstrap 5.3 (Using flexbox grids and UI layouts)
* **Typography & CDN Routing:** Google Fonts (Montserrat, Open Sans, Playfair Display via link fetching)
* **Version Control & Deployment:** Git/GitHub Pages (Continuous delivery from the source repository)
